# OriginalResourcePack

ギガンティック☆整地鯖で提供されている独自リソースパック（以下「本リソースパック」）です。  
本リソースパックの著作権は本サーバーOwnerのunchamaが保有しています。本リソースパックを無断で改変再配布したり二次配布したりすることを一切禁じます。

## 利用例

* 個人利用目的でDLして改変する→OK
* 改変したものをインターネット上へ公開する→NG
